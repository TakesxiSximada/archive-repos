* Development

** Start Emulator

Run the following command to launch the iOS emulator.

#+BEGIN_SRC shell
flutter emulators --launch apple_ios_simulator
#+END_SRC


** Start application

Run the following command to start application.

#+BEGIN_SRC
flutter run
#+END_SRC
